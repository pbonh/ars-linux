# zinstall Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a Bash bootstrapper (`zinstall`) that personalizes a freshly-installed Zirconium (Fedora bootc) system in one curl-pipe, and re-converges to the same state on every subsequent run.

**Architecture:** A single `install.sh` entry point sources phase modules under `lib/*.sh`, each exporting one `run_<phase>` function. Phases run in fixed order (preflight → brew → chezmoi → brewfile → distrobox → autostart → layered → system → summary). Every phase is independently idempotent and skippable via `--only`/`--skip`. State-changing commands flow through a `run_cmd` wrapper so `--dry-run` short-circuits execution while preserving control flow. Declarative inputs live under `packages/` (Brewfile, distroboxes.ini, autostart.list, layered.txt). Tests use `bats-core` with mocked external binaries placed on PATH.

**Tech Stack:** Bash 5+, `bats-core` for tests, Homebrew + `brew bundle` (with native `flatpak` directive), distrobox + `distrobox-assemble`, systemd `--user`, `rpm-ostree`, `bootc`, chezmoi, GitHub Actions for CI.

## Conventions for every task

- **TDD:** write the bats test first, watch it fail, implement minimally, watch it pass, then commit.
- **Bash style:** `#!/usr/bin/env bash` shebang on `install.sh` only. Library files have no shebang and start with `# shellcheck shell=bash`. Use `local` for function variables. Quote all expansions. Use `[[ ]]`, never `[ ]`.
- **Function naming:** phase entry points are `run_<phase>`. Internal helpers in libraries use `_<phase>__<name>` (e.g. `_autostart__slugify`).
- **Idempotency:** every state-changing command must be guarded by an explicit "is it already in the desired state?" check, except where the upstream tool guarantees idempotency (documented per phase in spec §7).
- **No destructive defaults:** `--prune` is the only path that removes things.
- **Commits:** one commit per task minimum. Conventional Commits format (`feat:`, `test:`, `fix:`, `chore:`, `docs:`).

# shellcheck shell=bash
# Stub external commands by writing scripts into $STUB_BIN. Each invocation
# appends its argv to $TEST_TMP/calls/<cmd>.log so tests can assert on it.

mock_cmd() {
  local name="$1"
  local exit_code="${2:-0}"
  local stdout="${3:-}"
  mkdir -p "$TEST_TMP/calls"
  cat >"$STUB_BIN/$name" <<EOF
#!/usr/bin/env bash
echo "\$@" >>"$TEST_TMP/calls/$name.log"
[[ -n "$stdout" ]] && printf '%s\n' "$stdout"
exit $exit_code
EOF
  chmod +x "$STUB_BIN/$name"
}

assert_called() {
  local name="$1"
  [[ -f "$TEST_TMP/calls/$name.log" ]] || {
    echo "expected $name to be called but it was not"
    return 1
  }
}

assert_not_called() {
  local name="$1"
  if [[ -f "$TEST_TMP/calls/$name.log" ]]; then
    echo "expected $name NOT to be called but it was: $(cat "$TEST_TMP/calls/$name.log")"
    return 1
  fi
}

assert_called_with() {
  local name="$1" expected="$2"
  grep -Fxq "$expected" "$TEST_TMP/calls/$name.log" || {
    echo "expected $name called with: $expected"
    echo "actual calls:"
    cat "$TEST_TMP/calls/$name.log"
    return 1
  }
}

# shellcheck shell=bash
# Logging primitives. All zinstall output flows through these.
# Honors NO_COLOR; emits plain text when stdout is not a TTY.

_log__color() {
  if [[ -n "${NO_COLOR:-}" ]] || [[ ! -t 1 ]]; then
    return 1
  fi
  return 0
}

_log__emit() {
  local stream="$1" tag="$2" color="$3" msg="$4"
  if _log__color; then
    printf '\e[%sm[%s]\e[0m %s\n' "$color" "$tag" "$msg" >&"$stream"
  else
    printf '[%s] %s\n' "$tag" "$msg" >&"$stream"
  fi
}

log::info()    { _log__emit 1 "INFO"  "36" "$*"; }
log::ok()      { _log__emit 1 "OK"    "32" "$*"; }
log::warn()    { _log__emit 2 "WARN"  "33" "$*"; }
log::error()   { _log__emit 2 "ERROR" "31" "$*"; }

log::section() {
  local title="$*"
  local rule
  rule="$(printf '=%.0s' {1..60})"
  if _log__color; then
    printf '\n\e[1;35m%s\n== %s\n%s\e[0m\n' "$rule" "$title" "$rule"
  else
    printf '\n%s\n== %s\n%s\n' "$rule" "$title" "$rule"
  fi
}

```bash
# --- Global flag state (defaulted by parse_args on every call) ---
UPGRADE=0
DRY_RUN=0
NO_REBOOT_PROMPT=0
PRUNE=0
VERBOSE=0
ONLY_PHASES=""
SKIP_PHASES=""

readonly ZINSTALL_PHASES="brew chezmoi brewfile distrobox autostart layered system"

usage() {
  cat <<'EOF'
Usage: install.sh [flags]

  --upgrade              Run brew bundle upgrade, chezmoi update, and bootc upgrade.
  --dry-run              Print every state-changing command, do not execute.
  --no-reboot-prompt     Suppress the reboot question; just report.
  --prune                Remove items no longer declared (layered/autostart/Brewfile).
  --only=<phase[,...]>   Run only listed phases. Names: brew, chezmoi, brewfile,
                         distrobox, autostart, layered, system.
  --skip=<phase[,...]>   Skip listed phases.
  -v, --verbose          Show every command before running it (set -x).
  -h, --help             Print usage and exit 0.
EOF
}

_validate_phase_list() {
  local list="$1" phase
  for phase in $list; do
    case " $ZINSTALL_PHASES " in
      *" $phase "*) ;;
      *) log::error "unknown phase: $phase"; return 1 ;;
    esac
  done
}

parse_args() {
  UPGRADE=0; DRY_RUN=0; NO_REBOOT_PROMPT=0; PRUNE=0; VERBOSE=0
  ONLY_PHASES=""; SKIP_PHASES=""
  while (($#)); do
    case "$1" in
      --upgrade)          UPGRADE=1 ;;
      --dry-run)          DRY_RUN=1 ;;
      --no-reboot-prompt) NO_REBOOT_PROMPT=1 ;;
      --prune)            PRUNE=1 ;;
      -v|--verbose)       VERBOSE=1 ;;
      --only=*)           ONLY_PHASES="${1#--only=}"; ONLY_PHASES="${ONLY_PHASES//,/ }" ;;
      --skip=*)           SKIP_PHASES="${1#--skip=}"; SKIP_PHASES="${SKIP_PHASES//,/ }" ;;
      -h|--help)          usage; return 0 ;;
      *) log::error "unknown option: $1"; usage >&2; return 2 ;;
    esac
    shift
  done
  if [[ -n "$ONLY_PHASES" && -n "$SKIP_PHASES" ]]; then
    log::error "--only and --skip are mutually exclusive"
    return 2
  fi
  [[ -n "$ONLY_PHASES" ]] && _validate_phase_list "$ONLY_PHASES"
  [[ -n "$SKIP_PHASES" ]] && _validate_phase_list "$SKIP_PHASES"
}

if [[ -z "${ZINSTALL_PARSE_ONLY:-}" ]]; then
  main() {
    parse_args "$@" || exit $?
    log::section "zinstall"
    log::info "argv parsed (upgrade=$UPGRADE dry-run=$DRY_RUN prune=$PRUNE)"
  }
  main "$@"
fi

```bash
# Wrap any state-changing command. Honors DRY_RUN (skip + log) and VERBOSE
# (echo the command before running). Always quotes argv when echoing.
run_cmd() {
  if (( VERBOSE )); then
    printf '+ %s\n' "$*" >&2
  fi
  if (( DRY_RUN )); then
    log::info "DRY-RUN: $*"
    return 0
  fi
  "$@"
}

# Retry $cmd up to $attempts times, doubling sleep between attempts.
# Honors ZINSTALL_RETRY_BASE_SLEEP for tests (default 1s).
retry_with_backoff() {
  local attempts="$1"; shift
  local sleep_for="${ZINSTALL_RETRY_BASE_SLEEP:-1}"
  local i=0 rc=0
  while (( i < attempts )); do
    if "$@"; then return 0; fi
    rc=$?
    i=$((i + 1))
    if (( i < attempts )); then
      log::warn "attempt $i/$attempts failed (exit $rc); retrying in ${sleep_for}s"
      sleep "$sleep_for"
      sleep_for=$((sleep_for * 2))
    fi
  done
  return "$rc"
}
```

# Refuses to proceed unless the host is a non-root, network-connected,
# Fedora-family bootc machine and sudo can be primed.

  printf '%s' "${ZINSTALL_OS_RELEASE_FILE:-/etc/os-release}"

  [[ -r "$file" ]] || return 1
  local id="" id_like=""
  # shellcheck disable=SC1090
  while IFS='=' read -r key val; do
    val="${val%\"}"; val="${val#\"}"
    case "$key" in
      ID) id="$val" ;;
      ID_LIKE) id_like="$val" ;;
    esac
  done <"$file"
  [[ "$id" == "fedora" ]] && return 0
  [[ "$id_like" == *fedora* ]] && return 0
  return 1
}

_preflight__sudo_keepalive() {
  (( ${ZINSTALL_KEEPALIVE_DISABLED:-0} )) && return 0
  ( while true; do sudo -n true; sleep 60; kill -0 "$$" 2>/dev/null || exit; done ) &
}

  if [[ "${EUID:-$(id -u)}" -eq 0 ]]; then
    log::error "refuse to run as root; zinstall must run as your normal user"

    return 1

    return 1

# shellcheck shell=bash
# Phase 4 — Distrobox: assemble missing containers from distroboxes.ini and
# enforce per-container exported_bins / exported_apps idempotently.

_distrobox__path() {
  printf '%s' "${ZINSTALL_DISTROBOXES_INI:-$ZINSTALL_DIR/packages/distroboxes.ini}"
}

# Print every [section] name from an INI file, one per line, in declaration order.
_distrobox__sections() {
  local file="$1" line
  while IFS= read -r line; do
    line="${line%$'\r'}"
    [[ "$line" =~ ^\[([^\]]+)\]$ ]] && printf '%s\n' "${BASH_REMATCH[1]}"
  done <"$file"
}

# Print the value of $key inside [$section] of $file, with surrounding quotes stripped.
_distrobox__field() {
  local section="$1" key="$2" file="$3" in_section=0 line val
  while IFS= read -r line; do
    line="${line%$'\r'}"
    if [[ "$line" =~ ^\[([^\]]+)\]$ ]]; then
      [[ "${BASH_REMATCH[1]}" == "$section" ]] && in_section=1 || in_section=0
      continue
    fi
    if (( in_section )) && [[ "$line" =~ ^[[:space:]]*${key}[[:space:]]*=[[:space:]]*(.*)$ ]]; then
      val="${BASH_REMATCH[1]}"
      val="${val%\"}"; val="${val#\"}"
      printf '%s' "$val"
      return 0
    fi
  done <"$file"
}

_distrobox__exists() {
  local name="$1"
  distrobox list 2>/dev/null | awk -F'|' 'NR>1 {gsub(/ /,"",$2); print $2}' | grep -Fxq "$name"
}

_distrobox__export_for() {
  local name="$1" file="$2" bins apps token
  bins="$(_distrobox__field "$name" "exported_bins" "$file")"
  apps="$(_distrobox__field "$name" "exported_apps" "$file")"
  for token in $bins; do
    run_cmd distrobox-export --container "$name" --bin "$token" || \
      log::warn "export bin failed for $name: $token"
  done
  for token in $apps; do
    run_cmd distrobox-export --container "$name" --app "$token" || \
      log::warn "export app failed for $name: $token"
  done
}

run_distrobox() {
  log::section "Phase 4 — Distrobox"

  local need
  for need in distrobox distrobox-assemble distrobox-export; do
    if ! command -v "$need" >/dev/null 2>&1; then
      log::error "$need not found; install distrobox before running zinstall"
      return 1
    fi
  done

  local file
  file="$(_distrobox__path)"
  if [[ ! -r "$file" ]]; then
    log::error "distroboxes.ini not readable: $file"
    return 1
  fi

  local name
  while IFS= read -r name; do
    [[ -z "$name" ]] && continue
    if _distrobox__exists "$name"; then
      log::ok "container '$name' already present; refreshing exports"
    else
      log::info "creating container '$name'"
      if ! run_cmd distrobox-assemble create --file "$file" --name "$name"; then
        log::error "failed to assemble container: $name"
        continue
      fi
    fi
    _distrobox__export_for "$name" "$file"
  done < <(_distrobox__sections "$file")
}

```bash
# shellcheck shell=bash
# Phase 5 — Autostart via systemd --user.
# Generates a zinstall-<slug>.service per entry in autostart.list. All managed
# units share the zinstall- prefix; user-authored units are never touched.

_autostart__list_path() {
  printf '%s' "${ZINSTALL_AUTOSTART_LIST:-$ZINSTALL_DIR/packages/autostart.list}"
}

_autostart__unit_dir() {
  printf '%s' "${ZINSTALL_USER_UNIT_DIR:-$HOME/.config/systemd/user}"
}

# Lowercase the basename and strip any extension/non-allowed chars.
_autostart__slugify() {
  local raw="$1" base
  base="${raw##*/}"
  base="${base%.*}"
  base="${base,,}"
  printf '%s' "$base" | tr -cs 'a-z0-9-' '-' | sed 's/^-//; s/-$//'
}

# Emit "slug|command" for each non-empty, non-comment line. Returns non-zero
# if any explicit slug fails the [a-z0-9-]+ rule.
_autostart__entries() {
  local file="$1" line slug cmd
  while IFS= read -r line; do
    line="${line%$'\r'}"
    [[ -z "${line// }" ]] && continue
    [[ "$line" =~ ^[[:space:]]*# ]] && continue
    if [[ "$line" == *"="* && "${line%%=*}" =~ ^[A-Za-z0-9_-]+$ ]]; then
      slug="${line%%=*}"
      cmd="${line#*=}"
      if [[ ! "$slug" =~ ^[a-z0-9-]+$ ]]; then
        printf 'invalid slug: %s\n' "$slug" >&2
        return 1
      fi
    else
      cmd="$line"
      slug="$(_autostart__slugify "${cmd%% *}")"
      if [[ -z "$slug" ]]; then
        printf 'could not derive slug from: %s\n' "$cmd" >&2
        return 1
      fi
    fi
    printf '%s|%s\n' "$slug" "$cmd"
  done <"$file"
}

_autostart__unit_body() {
  local slug="$1" cmd="$2"
  cat <<EOF
[Unit]
Description=zinstall-managed autostart: $slug
After=graphical-session.target
PartOf=graphical-session.target

[Service]
Type=simple
ExecStart=$cmd
Restart=on-failure
RestartSec=5

[Install]
WantedBy=graphical-session.target
EOF
}

_autostart__write_if_changed() {
  local path="$1" body="$2" tmp
  tmp="$(mktemp)"
  printf '%s' "$body" >"$tmp"
  if [[ -f "$path" ]] && cmp -s "$tmp" "$path"; then
    rm -f "$tmp"
    return 1   # unchanged
  fi
  if (( DRY_RUN )); then
    log::info "DRY-RUN: would write $path"
    rm -f "$tmp"
    return 0
  fi
  mv "$tmp" "$path"
  return 0   # written
}

run_autostart() {
  log::section "Phase 5 — Autostart"
  local list dir
  list="$(_autostart__list_path)"
  dir="$(_autostart__unit_dir)"
  mkdir -p "$dir"

  if [[ ! -r "$list" ]]; then
    log::warn "autostart list not readable: $list (skipping)"
    return 0
  fi

  local declared_slugs=() entries entry slug cmd unit reload_needed=0
  if ! entries="$(_autostart__entries "$list")"; then
    log::error "autostart.list parse error"
    return 1
  fi

  while IFS= read -r entry; do
    [[ -z "$entry" ]] && continue
    slug="${entry%%|*}"
    cmd="${entry#*|}"
    declared_slugs+=("$slug")
    unit="$dir/zinstall-$slug.service"
    if _autostart__write_if_changed "$unit" "$(_autostart__unit_body "$slug" "$cmd")"; then
      reload_needed=1
      log::ok "wrote $(basename "$unit")"
    fi
  done <<<"$entries"

  if (( PRUNE )); then
    local f base keep
    while IFS= read -r f; do
      [[ -z "$f" ]] && continue
      base="$(basename "$f" .service)"
      base="${base#zinstall-}"
      keep=0
      for slug in "${declared_slugs[@]}"; do
        [[ "$slug" == "$base" ]] && keep=1 && break
      done
      if (( ! keep )); then
        run_cmd systemctl --user disable --now "zinstall-$base.service" || true
        run_cmd rm -f "$f"
        reload_needed=1
        log::ok "pruned zinstall-$base.service"
      fi
    done < <(find "$dir" -maxdepth 1 -type f -name 'zinstall-*.service' 2>/dev/null)
  fi

  if (( reload_needed )); then
    run_cmd systemctl --user daemon-reload
  fi

  for slug in "${declared_slugs[@]}"; do
    run_cmd systemctl --user enable --now "zinstall-$slug.service"
  done
}

# shellcheck shell=bash
# Phase 6 — rpm-ostree layered packages.
# Diffs packages/layered.txt against the current deployment's requested-packages
# and applies install/uninstall in single batched calls. Sets REBOOT_NEEDED on change.

_layered__path() {
  printf '%s' "${ZINSTALL_LAYERED_TXT:-$ZINSTALL_DIR/packages/layered.txt}"
}

_layered__parse_list() {
  local file="$1" line
  while IFS= read -r line; do
    line="${line%$'\r'}"
    line="${line#"${line%%[![:space:]]*}"}"
    line="${line%"${line##*[![:space:]]}"}"
    [[ -z "$line" ]] && continue
    [[ "$line" =~ ^# ]] && continue
    printf '%s\n' "$line"
  done <"$file"
}

_layered__current() {
  rpm-ostree status --json 2>/dev/null \
    | jq -r '.deployments[0]["requested-packages"][]' 2>/dev/null \
    | sort -u
}

run_layered() {
  log::section "Phase 6 — rpm-ostree layered packages"
  local file
  file="$(_layered__path)"
  if [[ ! -r "$file" ]]; then
    log::warn "layered.txt not readable: $file (skipping)"
    return 0
  fi

  local desired current to_add to_remove
  desired="$(_layered__parse_list "$file" | sort -u)"
  current="$(_layered__current)"
  to_add="$(comm -23 <(printf '%s\n' "$desired") <(printf '%s\n' "$current"))"
  to_remove="$(comm -13 <(printf '%s\n' "$desired") <(printf '%s\n' "$current"))"

  if [[ -n "$to_add" ]]; then
    log::info "installing layered: $(echo "$to_add" | tr '\n' ' ')"
    # shellcheck disable=SC2086
    if run_cmd sudo rpm-ostree install --idempotent --allow-inactive $to_add; then
      REBOOT_NEEDED=1
    else
      log::error "rpm-ostree install failed"
      return 1
    fi
  else
    log::ok "no packages to add"
  fi

  if [[ -n "$to_remove" ]]; then
    if (( PRUNE )); then
      log::info "removing layered: $(echo "$to_remove" | tr '\n' ' ')"
      # shellcheck disable=SC2086
      if run_cmd sudo rpm-ostree uninstall $to_remove; then
        REBOOT_NEEDED=1
      else
        log::error "rpm-ostree uninstall failed"
        return 1
      fi
    else
      log::warn "extra layered packages present (use --prune to remove): $(echo "$to_remove" | tr '\n' ' ')"
    fi
  fi
}

  log::section "Summary"

should_run() {

  if [[ -n "$ONLY_PHASES" ]]; then

  fi
  if [[ -n "$SKIP_PHASES" ]]; then

if [[ -z "${ZINSTALL_PARSE_ONLY:-}" ]]; then
  main() {
    parse_args "$@" || exit $?
    (( VERBOSE )) && set -x
    _install__init_runlog
    log::section "zinstall"
    run_preflight || { log::error "preflight failed; aborting"; exit 1; }

    _run_phase brew      run_brew
    _run_phase chezmoi   run_chezmoi
    _run_phase brewfile  run_brewfile
    _run_phase distrobox run_distrobox
    _run_phase autostart run_autostart
    _run_phase layered   run_layered
    _run_phase system    run_system

    print_summary
    maybe_reboot
    if any_failed; then exit 1; fi
  }
  main "$@"
fi

# zinstall

Bring a freshly installed Zirconium (Fedora bootc + Niri) box to a fully
personalized state with one command, and re-converge to that state on every
subsequent run.

## One-shot bootstrap

```bash
curl -fsSL https://raw.githubusercontent.com/pbonh/zinstall/main/install.sh | bash
```

## Re-runs

```bash
git clone https://github.com/pbonh/zinstall.git ~/.local/share/zinstall
cd ~/.local/share/zinstall
./install.sh                      # converge
./install.sh --upgrade            # converge + upgrade brew/chezmoi/bootc
./install.sh --prune              # also remove items no longer declared
./install.sh --only=autostart     # run a single phase
./install.sh --dry-run            # show what would change
./install.sh --help
```

## Phases

| # | Phase     | Owns                                                  |
|---|-----------|-------------------------------------------------------|
| 0 | preflight | bootc/Fedora/network/sudo checks                      |
| 1 | brew      | Install Homebrew if missing                           |
| 2 | chezmoi   | Pull dotfiles from `pbonh/zdots`                      |
| 3 | brewfile  | Apply `packages/Brewfile` (formulae + Flatpaks)       |
| 4 | distrobox | Assemble containers from `packages/distroboxes.ini`   |
| 5 | autostart | Generate `~/.config/systemd/user/zinstall-*.service`  |
| 6 | layered   | Reconcile `packages/layered.txt` against rpm-ostree   |
| 7 | system    | `bootc upgrade` (only with `--upgrade`)               |

## Configuration files

| File                       | Format                                |
|----------------------------|---------------------------------------|
| `packages/Brewfile`        | Homebrew Bundle (`brew`, `flatpak`…) |
| `packages/distroboxes.ini` | `distrobox-assemble` INI              |
| `packages/autostart.list`  | One per line; `slug=cmd` or bare cmd  |
| `packages/layered.txt`     | One rpm-ostree package per line       |

See `spec.md` for the full design.

## Tests

```bash

```
